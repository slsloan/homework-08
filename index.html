ABOUT ME

Since I was a kid, I have always enjoyed being creative - from drawing to building worlds with Legos - I discovered that I could combine creativity with technology in video games.

After undergrad, I explored careers in the tech industry and took a brief weekend course on coding. After that class, I realized I couldn’t put it down and wanted to expand my knowledge on it. That’s when I decided to enroll in DU’s Full Stack Web Development Bootcamp.

During my bootcamp experience, I was in charge of managing a lot of the back-end data in group projects. I really enjoy working with MySQL, ORM's, and MongoDB. I'm working on sharpening my skills with relationship databases so that I can pursue a career where I can work closer with data. I have done projects in the past that showcase my love for data and organization such as a virtual behavioural tracking system for employees. While working at Gusto, I've developed a metrics tracking spreadsheet that shows information regarding each individual representative’s sales performance that week.

Work Day Scheduler

Created a simple calendar application that allows the user to save events for each hour of the day. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery. The events save to your local storage so when the page is refreshed you will still have the same events so you can access it again. Also, we utilize color coding each time block to show you whether the time is either current, future, or past. Deployed here!
